Hi nama saya Muhammad Ghazy Humaidi, anda bisa memanggil saya Ghazy, saya sekolah di pondok pesantren Daarul Arqam Muhammadiyah Daerah Garut dan saya duduk di kelas 3 Tsanawiyah / kelas 3 SMP, saat website ini di buat usia saya 15 Tahun dan dibuat pada Bulan Maret ketika Virus Vaksin Covid-19 sudah mulai berdatangan

Ngomong ngomong tentang Covid-19, saya memulai menjadi seorang Web Developer pada saat Covid-19 karena saya menyadari bahwa waktu akan terus berjalan sementara saya hanya diam di rumah dengan kegiatan yang lebih sedikit. Maka untuk mengisi waktu luang saya coba untuk belajar menjadi seorang Web Developer. Saya sudah belajar menjadi seorang Web Developer selama kurang lebih 7 Bulan

Cita cita saya adalah sebagai Bug Hunter dan Cyber Security, saya belajar Web Developer karena untuk belajar dasar dasar yang dibutuhkan untuk menggapai cita cita saya. Setelah Webssite ini beres saya ingin belajar Bahasa Pemograman Javascript, setelah itu lalu saya akan fokus ke Bug Hunter dan Cyber Security karena memiliki dasar dari Javascript

Selain bejalar hal baru yaitu menjadi Web Developer, Pada saat pandemi Covid-19, saya melakukan kegiatan dalam Orgnanisasi Tapak Suci. Tapak Suci adalah salah satu Orgnanisasi yang berada di dalam naungan Muhammadiyah yang bergerak dalam bidang Bela Diri.

Dikarenakan Orgnanisasi ini yang bergerak dalam bidang bela diri dan banyak kegiatan yang harusnya di lakukan di luar ruangan dengan banyak orang namun sekarang malah kita harus diam di Dalam rumah dan sendiri sendiri, banyak acara yang dibatalkan dan akhirnya harus membuat acara yang sebelumnya belum pernah dilakukan, sehingga hasilnya pun kurang maksimal karena ini adalah first Impression kita dalam membuat acara secara Daring
